# Mobile UI Stylist Skill 📱✨

> **An AI-Powered Mobile Interface Customization Skill.**
> 这是一个用于美化手机 UI 的 AI Agent 技能。

## 🚀 功能特点
- **智能配色**：提供精准的 HEX 色值方案。
- **布局指南**：针对 iOS/Android 的网格化布局建议。
- **工具推荐**：适配 Widgy, KWGT, Nova 等主流美化工具。

## 📂 仓库结构
- `SKILL.md`: 核心 Agent 指令集（用于导入 Claw Hub）。
- `README.md`: 项目使用说明。

## 🛠 如何使用
1. 下载本仓库文件。
2. 在 [Claw Hub](https://hub.claw.ai) 上传包含 `SKILL.md` 的文件夹。
3. 开始定制你的专属手机桌面！
